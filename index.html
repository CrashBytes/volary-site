Built different

🔒

Encrypted Vault

XChaCha20-Poly1305 encryption with Argon2id key derivation. Your passwords and sensitive data stay locked down.

🚫

Tracker Blocking

Built-in network filter blocks ads and trackers before they load. No extensions needed.

📖

Reading Mode

Strip away clutter and read articles in clean typography. Semantic content extraction, not just CSS hacks.

♿

Accessible

Full keyboard navigation, ARIA markup, screen reader optimization, colorblind filters, and reduced motion support.

🛠

Crash Recovery

Session state is continuously saved. If something goes wrong, your tabs come back exactly where you left off.

👁

Open Source

Every line of code is auditable. No telemetry, no tracking, no surprises. Built in the open.
